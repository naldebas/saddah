=== Useful Commands ===

View all resources:
  kubectl get all -n {{ include "saddah.namespace" . }}

View logs:
  kubectl logs -n {{ include "saddah.namespace" . }} -l app.kubernetes.io/component=api -f
  kubectl logs -n {{ include "saddah.namespace" . }} -l app.kubernetes.io/component=web -f

Port forward for local access:
  kubectl port-forward -n {{ include "saddah.namespace" . }} svc/{{ .Values.api.name }} 3000:{{ .Values.api.service.port }}
  kubectl port-forward -n {{ include "saddah.namespace" . }} svc/{{ .Values.web.name }} 8080:{{ .Values.web.service.port }}

=== Health Checks ===

API Health:
  kubectl exec -n {{ include "saddah.namespace" . }} deploy/{{ .Values.api.name }} -- curl -s localhost:{{ .Values.api.service.targetPort }}/api/v1/health

Web Health:
  kubectl exec -n {{ include "saddah.namespace" . }} deploy/{{ .Values.web.name }} -- curl -s localhost:{{ .Values.web.service.targetPort }}/health
